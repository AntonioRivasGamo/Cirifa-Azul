# Cirifa Azul
## A pet adoption platform with a social-media-style experience
Cirifa Azul is a web application designed to connect people who want to adopt or put a pet up for adoption. It works similarly to a social network, allowing users to create profiles, publish adoption posts, and explore available pets through advanced search filters to find the perfect match.

### Key Features
* User Accounts: Create an account to adopt pets or publish pets for adoption.
* Pet Listings: Upload photos, add detailed information, and manage your adoption posts.
* Advanced Search Filters: Search by species, size, age, location, and more to find the most suitable pet.
* Social Features: Explore profiles, save favorites, and contact pet owners or shelters.
